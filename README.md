# üé≠ Predict the Mood

A machine learning-powered application that analyzes text input and predicts the emotional mood of the user. The project combines NLP-based feature extraction with multiple trained classifiers to deliver accurate mood detection through an interactive interface.

---

## ‚ú® Features

- **Multi-Model Mood Prediction** ‚Äî Uses four trained classifiers (Logistic Regression, Naive Bayes, Decision Tree, KNN) to predict mood from text input
- **TF-IDF Text Vectorization** ‚Äî Converts raw text into meaningful numerical features using a pre-trained TF-IDF model
- **Pre-trained Model Support** ‚Äî All models are serialized as `.pkl` files for instant inference without retraining
- **Interactive Deployment Interface** ‚Äî Powered by `deploy.py` for a clean, real-time user experience
- **Comparative Model Evaluation** ‚Äî Easily compare predictions across different classifiers
- **Lightweight & Modular** ‚Äî Clean project structure makes it easy to swap models or extend functionality
- **Fast Inference Pipeline** ‚Äî End-to-end prediction in milliseconds from text input to mood output

## ‚öôÔ∏è Installation & Setup

**1. Clone the repository**

```bash

cd Predict-the-Mood
```

**2. Create a virtual environment**

```bash
python -m venv venv
```

**3. Activate the virtual environment**

```bash

venv\Scripts\activate
```

**4. Install dependencies**

```bash
pip install -r requirements.txt
```

**5. Run the application**

```bash
streamlit run deploy.py
```

### Step-by-Step Flow

1. **User Input** ‚Äî The user enters a text string (sentence, phrase, or paragraph) into the Streamlit interface.
2. **Text Preprocessing** ‚Äî The raw input is cleaned: lowercased, stripped of punctuation and noise.
3. **TF-IDF Vectorization** ‚Äî The pre-trained `tfidf_model.pkl` transforms the cleaned text into a sparse feature vector representing word importance.
4. **Parallel Model Inference** ‚Äî The feature vector is passed to all four pre-trained classifiers simultaneously:
   - `lr_model.pkl` ‚Üí Logistic Regression
   - `nb_model.pkl` ‚Üí Naive Bayes
   - `dt_model.pkl` ‚Üí Decision Tree
   - `knn_model.pkl` ‚Üí K-Nearest Neighbors
5. **Mood Prediction** ‚Äî Each model outputs a predicted mood label based on learned patterns from the training data.
6. **Result Display** ‚Äî The predicted mood(s) are displayed back to the user via the Streamlit interface.
